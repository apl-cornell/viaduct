class Replication(hosts: Set<Host>) : Plaintext
Content copied to clipboard

The protocol that replicates data and computations across a set of hosts in the clear.

Replication increases integrity, but doing it in the clear sacrifices confidentiality. Additionally, availability is lost if any participating host aborts.
