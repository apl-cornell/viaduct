bindsTighterThan

fun Operator.bindsTighterThan(other: Operator): Boolean
Content copied to clipboard

Returns true when this operator has precedence higher than or equal to other.

This function is reflexive, that is, x.bindsTighterThan(x) returns true. However, it is not necessarily transitive, and it does not order all operators.

Two operators x and y have the same precedence if x.bindsTighterThan(y) and y.bindsTighterThan(x).
